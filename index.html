Projects

The Boxers with Parkinson's Project

IThe Boxers with Parkinson's app stores all boxer data and assessment results digitally, replacing the need for cumbersome paper files and speeding up the assessment process. It's easy to use and accessible to boxers of all physical levels. The app uses a foundation of HTML, CSS, and Javascript and uses Cordova to convert the code into an Android application to be used on tablets. The Boxers with Parkinson's app is currently an Android only application and is going live in December 2019.

Developed with: Node.js, Express.js, Cordova, MongoDB, HTML, CSS, Bootstrap, Javascript/jQuery, JSON

WTIA Connections

A web application that allows volunteer tech industry speakers to connect with high school teachers/counselors and request speaking opportunities. Set to help inspire students from over 200 high schools in Washington State. Still in the developmental stage.

Developed with: HTML, CSS, Bootstrap, Javascript/jQuery, PHP and MySQL

The Ice Cream Shoppe

A mini web application that would give an ice cream shop the means to take customer orders and allow employees to view and fulfill those orders. Submitted orders are sent to an SQL database using PHP then queried and displayed on the employee's view order page.

Developed with: HTML, CSS, Bootstrap, Javascript/jQuery, PHP and MySQL
